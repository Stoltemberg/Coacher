# Roadmap de Implementação de SEO (Fases)

## Fase 1: Fundação Técnica (Concluída ✅)
- [x] Adição de Metadatas Dinâmicos no Next.js (layout.tsx e page.tsx).
- [x] Implementação de Schema.org JSON-LD (SoftwareApplication).
- [x] Criação de `sitemap.xml` dinâmico.
- [x] Criação de `robots.txt`.
- [x] Correção de hierarquia de Headings (H1 "sr-only").

## Fase 2: Estruturação do Blog (Semanas 1-2)
- [ ] Criar rota `/blog` no Next.js (ex: `app/blog/page.tsx`).
- [ ] Integrar um CMS headless (MDX local, Contentlayer ou Sanity) para facilidade de postagem.
- [ ] Implementar Schema `Article` e `BreadcrumbList` nas páginas de blog.

## Fase 3: Criação de Conteúdo Escalável (Mês 1-3)
- [ ] Lançar 2 artigos por semana baseados no Content Calendar.
- [ ] Gerar páginas dinâmicas para campeões (ex: `/campeoes/yasuo`) que puxam dados da própria IA do Coacher e se rankeiam para "Como jogar de Yasuo".

## Fase 4: Autoridade e Link Building (Mês 4+)
- [ ] Parcerias com streamers e YouTubers menores.
- [ ] Criar ferramentas gratuitas (ex: "Calculadora de MMR" ou "Tier List do Patch") e hospedar sob o domínio para gerar backlinks naturais.
